1. Установитье node.js если не установлен. Для этого следует запустить файл "node-v20.10.0-x64.msi" с выбором параметров по умолчанию (ПОТРЕБУЕТСЯ ПЕРЕЗАГРУЗКА)

2. Для старта скрита нажмите двойны щелчком на файл "run.bat" - после открытия появится консоль, отображающая процесс парсинга. Все новые данные будут по умолчанию сохранятся в папку "output".

Для настройки ПАРАМЕТРОВ парсинга следует открыть файл ".env" (открывается в любом текстовом редакторе)

RETRY_COUNT=10        - Количество повторений при ошибке сервера
TIMEOUT=0             - Таймаут запроса (0 и меньше - без таймаута)
USE_PROXY=0           - Использовать прокси (файл "proxy.txt") (0 - не использовать, 1 - использовать)
OUTPUT_PATH=./output  - Относительный путь для сохранения данных

Для изменений ФИЛЬТРОВ парсина следует изменить файл "data.json".
По ключу fromData необходимо вписать ключи для поиска.
(Сбор фильтров будет автоматизирован в расширении)
